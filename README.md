# Layanan IT

Web Profile dilengkapi dengan pendaftaran customer yang terdiri dari tiga jenis akses pengguna: Admin, Pendaftar, dan Programmer.

## Pendaftar
- Ketika pendaftar mendaftar di website, admin akan melakukan approval.
- Setelah approval, sistem akan mengirimkan username dan password login serta invoice ke email dan WhatsApp pendaftar.
- Pendaftar dapat melihat progress pengerjaan, rincian pembayaran, melakukan pembayaran cicilan, dan mengelola profil mereka.

## Programmer (Back End)
- Programmer dapat melihat job desk mereka dan mengisi progress pengerjaan secara online.
- Progress yang diisi oleh programmer dapat dilihat oleh pendaftar.

## Admin (Back End)
- Admin dapat melakukan approval pendaftar.
- Admin dapat menambah, menghapus, dan mengedit data pendaftar secara manual.
- Admin dapat menambah, menghapus, dan mengedit job desk project programmer.
- Admin dapat menambah, menghapus, dan mengedit data programmer.
- Admin dapat mengatur akses pengguna untuk programmer dan pendaftar.
- Admin dapat melakukan approval pembayaran.
